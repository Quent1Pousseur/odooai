### Table 2 — Engineering Backend + AI

→ *HR Director note : Junior (20) se sent mis de cote — a integrer davantage. Backend Arch et Senior Backend alignes sur les faiblesses techniques. Odoo Expert a un point critique sur les workflows — feature manquante. Prompt Eng frustre par l'absence d'evals.*
