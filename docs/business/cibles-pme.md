# OdooAI — Liste PME Cibles pour Interviews

## Criteres de selection
- PME francophone (France, Belgique, Suisse)
- 5-100 employes
- Utilise Odoo depuis > 6 mois
- Au moins 3 modules actifs
- Pas un integrateur Odoo (utilisateur final)

## Canaux de sourcing

### Canal 1 : Forums Odoo
- forum.odoo.com — filtrer par questions francophones recentes
- Les utilisateurs qui posent des questions = ceux qui veulent faire plus avec Odoo

### Canal 2 : LinkedIn
- Recherche : "Odoo" + "PME" + pays francophone
- Groupes : "Odoo Francophone", "Odoo Community", "ERP pour PME"
- Cibler les profils : CEO, COO, responsable IT de PME

### Canal 3 : Integrateurs partenaires
- **Camptocamp** (Suisse) — specialiste Odoo, centaines de clients PME
- **Acsone** (Belgique) — integrateur historique, focus manufacturing
- **Smile** (France) — grand integrateur open source
- Approche : "Nous developons un outil qui aide VOS clients a mieux utiliser Odoo. Pouvez-vous nous mettre en contact avec 2-3 clients volontaires ?"

### Canal 4 : Communaute OCA
- Odoo Community Association — meetups, listes de diffusion
- Les contributeurs OCA connaissent des PME utilisatrices

### Canal 5 : Annuaires
- odoo.com/partners — liste publique des partenaires et references
- societe.com / linkedin — identifier les PME francophones

## Approche de contact

### Message type (LinkedIn/email)
```
Bonjour [Prenom],

Je developpe un assistant IA pour les utilisateurs d'Odoo.
L'idee : l'IA analyse votre configuration et vous montre ce
qu'Odoo peut faire que vous n'utilisez pas encore.

J'aimerais avoir votre avis (15-20 min de votre temps) :
- Est-ce un vrai probleme pour vous ?
- Qu'est-ce que vous aimeriez qu'un tel outil fasse ?

Pas de vente, juste ecouter. Votre retour est precieux.

[Nom]
Fondateur, OdooAI
```
