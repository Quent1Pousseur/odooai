## BLOC 3 — Engineering

### Backend Architect (08)
- **Plan Orchestrator** : "Je propose un design simple pour Sprint 1 :
  1. `agents/orchestrator.py` : recoit une question, detecte le domaine, charge le BA Profile
  2. `agents/ba_agent.py` : construit le prompt avec le BA Profile + la question, appelle le LLM
  3. `cli.py` : ajouter commande `odooai chat` — boucle input/output
  Le Guardian est wire entre l'Orchestrator et les appels Odoo."

> **Senior Backend Dev (19)** : "Simple et testable. Pas de over-engineering."

  5. Inclure le disclaimer"
