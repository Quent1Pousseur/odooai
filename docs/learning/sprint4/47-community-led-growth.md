# Learning — Community Manager (47) — Community-Led Growth
## Date : 2026-03-20
## Duree : 4 heures

## Ce que j'ai appris

### Community-Led Growth (CLG) vs Product-Led Growth (PLG)
- PLG : le produit se vend tout seul (freemium, viral loop)
- CLG : la communaute genere la confiance, les referrals, et le feedback
- Pour OdooAI, c'est CLG qui fait sens : les PME font confiance aux recommandations de pairs, pas aux pubs

### L'ecosysteme Odoo — ou sont les communautes
| Canal | Taille estimee | Pertinence OdooAI |
|-------|---------------|-------------------|
| community.odoo.com (forum officiel) | 500K+ posts | Haute — questions fonctionnelles |
| OCA GitHub (Odoo Community Association) | 2500+ repos | Moyenne — technique |
| LinkedIn "Odoo" | 200K+ followers | Haute — decision makers |
| Reddit r/odoo | 5K members | Moyenne — early adopters |
| Odoo Experience (event annuel) | 3000+ participants | Tres haute — partners + clients |
| Groupes Facebook Odoo | 50K+ membres | Haute — PME francophones |

### Strategie pour OdooAI
1. **Phase 1 (maintenant)** : Observer et comprendre — lire les questions recurrentes sur le forum, identifier les pain points
2. **Phase 2 (beta)** : Contribuer avec valeur — repondre aux questions en citant des fonctionnalites meconnues (exactement ce que fait OdooAI)
3. **Phase 3 (lancement)** : Creer notre propre communaute — Discord/Slack pour les beta users
4. **Phase 4 (croissance)** : Ambassadeurs — les PME satisfaites recommandent OdooAI

### Les 3 erreurs a eviter
1. **Spam** : poster "essayez OdooAI !" partout → ban immediat
2. **Ignorer le feedback negatif** : chaque critique est une opportunite
3. **Pas de valeur avant la vente** : donner des insights gratuits AVANT de vendre le produit

## Comment ca s'applique a OdooAI

- Les forums Odoo sont remplis de questions que OdooAI peut resoudre ("comment activer X ?")
- LinkedIn est le canal #1 pour toucher les decideurs PME francophones
- L'Odoo Experience 2026 est une cible — si on a un produit stable, on peut y aller
- Le contact BizDev avec l'integrateur Bruxelles est exactement du CLG
