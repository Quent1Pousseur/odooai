# OdooAI — Guide de Demarrage en 5 Minutes
## Tech Writer (29) + Customer Success (17)
## Date : 2026-03-21

---

## Etape 1 — Ouvrez OdooAI (30 secondes)

Rendez-vous sur **odooai.com** et connectez-vous a votre compte.

Vous arrivez sur le chat. C'est ici que tout se passe.

---

## Etape 2 — Connectez votre Odoo (2 minutes)

1. Cliquez sur **"Connecter Odoo"** en haut a droite
2. Remplissez les 4 champs :
   - **URL** : l'adresse de votre Odoo (ex: https://mon-entreprise.odoo.com)
   - **Base de donnees** : le nom de votre base
   - **Login** : votre email Odoo
   - **Cle API** : generee dans Odoo (voir ci-dessous)
3. Cliquez **"Connecter"**

**Comment obtenir votre cle API :**
Dans Odoo → Parametres → Utilisateurs → votre profil → Cles API → Nouvelle cle API → copiez la cle.

Quand l'indicateur passe au **vert**, votre Odoo est connecte.

---

## Etape 3 — Posez votre premiere question (1 minute)

Tapez votre question en francais dans la barre en bas. Exemples :

- *"Quelles fonctionnalites de stock je n'utilise pas ?"*
- *"Comment automatiser mes relances de paiement ?"*
- *"Ma configuration de vente est-elle optimale ?"*

OdooAI analyse votre configuration et repond en quelques secondes.

---

## Etape 4 — Lisez la reponse (1 minute)

Chaque reponse contient :
- Un **diagnostic** de votre situation
- Des **recommandations** concretes avec le niveau de difficulte
- Le **chemin dans Odoo** pour activer chaque fonctionnalite
- Les **sources** (modules et modeles utilises)

---

## Etape 5 — Continuez la conversation (30 secondes)

Posez des questions de suivi :
- *"Comment activer cette fonctionnalite ?"*
- *"Quel est le risque si j'active ca ?"*
- *"Montre-moi les details de mes commandes"*

OdooAI se souvient du contexte de la conversation.

---

## Bon a savoir

- OdooAI fonctionne en **lecture seule** — il ne modifie jamais vos donnees
- Les donnees sensibles (RH, salaires) sont **anonymisees** automatiquement
- Chaque reponse cite ses **sources** — verifiez toujours avant d'agir
- OdooAI ne remplace PAS un expert-comptable ou un avocat
